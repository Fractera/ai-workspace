# docs/platforms/

Documentation for AI coding platforms supported by Fractera.

Each subdirectory contains documentation files for one platform, sourced from official docs.

## Platforms

- [kimi-code/](kimi-code/) — Kimi Code (Moonshot AI)
- [qwen-code/](qwen-code/) — Qwen Code (Alibaba)
- [claude-code/](claude-code/) — Claude Code (Anthropic)
- [gemini-cli/](gemini-cli/) — Gemini CLI (Google)
- [codex/](codex/) — Codex CLI (OpenAI)
- [open-code/](open-code/) — Open Code
